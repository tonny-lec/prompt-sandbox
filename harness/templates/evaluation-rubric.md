# Evaluation Rubric

## Criteria

| Criterion | What Good Looks Like |
| --- | --- |
| Goal fit | 出力が goal を外していない |
| Clarity | 次の行動が明確 |
| Handoff quality | 別 run が継続できる |
| Artifact completeness | 必須 artifact が揃っている |
| Risk visibility | リスクが記録されている |

## Scoring

- 5: すぐ流用できる
- 4: 小さな修正で流用できる
- 3: 方向性はよいが不足が目立つ
- 2: 重要情報が欠ける
- 1: 実験として成立していない
