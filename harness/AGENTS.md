# AGENTS.md

## Purpose

この実験場で作業する agent のふるまいを短く定義する。

## Default Rules

- まず `docs/` を読む
- 実験では artifact を `runs/` に残す
- 長時間タスクでは handoff を構造化して残す
- prompt の変更と scenario の変更は同時に行わない
- ルートの repo 文脈ではなく、この実験場の文脈を優先する
